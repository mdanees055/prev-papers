# College PYQ Database

## Description
This project contains the database schema for storing previous year question papers.

## Tables
- Subject: Stores subject details
- Paper: Stores exam papers with year and type(mid,sem)

## Features
- Foreign key relationship between Subject and Paper
- Unique constraint on (subject_code, year, exam_type)
- Sample data included

## How to Use
1. Open MySQL Workbench
2. Run the file college_pyq.sql
3. Database and tables will be created automatically
